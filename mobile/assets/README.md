# App Assets

This folder contains the app icons and splash screens for iOS and Android.

## Required Assets

### For Production Apps:

1. **icon.png** - 1024x1024px app icon
2. **adaptive-icon.png** - 1024x1024px Android adaptive icon
3. **splash.png** - 2048x2048px splash screen
4. **favicon.png** - 48x48px web favicon (if using web)

## Temporary Placeholder

Currently using placeholder assets. For production, replace with actual branded icons.

To generate icons automatically, you can use:
```bash
npx expo install expo-splash-screen
npx expo install expo-asset
```

Or use online tools like:
- https://appicon.co/
- https://www.figma.com/
